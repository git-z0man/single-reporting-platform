# SRP Manufacturer's Guide — public build

Live site: deployed via GitHub Pages from `main`.

This repository holds **the public build only**. Edits to `index.html`,
`styles.css`, or `app.js` here will be overwritten by the next build.

The full version (including portal screenshots and information shared
through the CRA Expert Group) is maintained privately and not distributed
through this repo. The public build was sanitised per Commission feedback:
non-public information has been removed.

Not affiliated with ENISA. Unofficial guide based on the Commission's
draft CRA guidance (not yet adopted). Not legal advice.
